CSS Flex(Flexible Box) 설명


참고
대부분 사이트 : 레이아웃 수직 구성. 위~아래로 스크롤하여 사용
레이아웃이 기본적으로 위~아래로 수직적으로 쌓이기 때문에 쉽게 만들 수 있다.
하지만 수평 구조. 왼쪽~오른쪽으로의 레이아웃은 만들기가 어렵다.
하지만 flex를 이용하면 만들기가 쉽다.
flex는 요소의 크기가 불분명하거나 동적인 경우에도 각 요소를 정렬할 수 있는 효율적인 방법을 제공한다.

flex의 개념


  • container : items를 감싸는 부모 요소 -> 사용 가능 속성 : display, flex-flow, justify-content
  • items : container에 감싸져 있는 자식 요소 -> 사용 가능 속성 : order, flex, align-self

주축(main-axis), 교차축(cross-axis)의 개념

주축 : items이 쌓이는 방향 -> justify-content
교차축 : 주축과 수직인 방향 -> align-items
즉, 방향(수평, 수직)에 따라 주축, 교차축이 달라진다
