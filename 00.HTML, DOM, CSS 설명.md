HTML : Hyper Text Markup Language
- 문서를 작성하기 위한 규칙
- 프로그래밍 언어 X
- HTML의 종류
  1. BLOCK Tag : 브라우저 창의 좌측~우측까지 전체 길이를 다 차지 (컨텐츠 길이와 상관X)
  - ex) div, p
  2. Inline Tag : 컨텐츠 길이만큼만 차지
  - ex) span
- DOM (Document Object Model) : 문서 구조를 그림으로 구조화된 모습으로 표현하는 방식
- 주석 : //, <!-- -->

CSS
- HTML에 style 적인 효과를 주기 위한 규칙
- 프로그래밍 언어 X
- 주석 : /* */
