div는 기본적으로 BLOCK Tag. 한 줄을 통째로 차지한다.
BLOCK Tag, Inline Tag 둘 다 포함 가능하다.
자동으로 앞뒤 줄바꿈이 된다.
ex) h1~h6(글씨 크기), p(문단), ul(● 목록), ol(1. 2. 3. 목록)
span은 기본적으로 Inline Tag. 콘텐츠 길이만큼 차지한다.
Inline Tag는 반드시 BLOCK Tag 안에 들어가 있어야 하고, BLOCK Tag를 담을 수 없다.
줄바꿈이 되지 않는다.
ex) img(이미지), input(텍스트 입력, 제출 버튼, 체크 박스 등등), a(링크)
